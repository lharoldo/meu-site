🌐 Sobre o Projeto

Este site apresenta iniciativas abertas desenvolvidas por LH para conectar dispositivos, automatizar redes e explorar tecnologias como blockchain e IPv6 de forma acessível e educativa.

📡 Internet das Coisas (IoT)

Monitoramento de sensores, automação residencial, comunicação via MQTT e integração com plataformas abertas.

🧬 IPv6

O projeto incentiva o uso do IPv6 para garantir escalabilidade, segurança e conectividade em redes modernas. Todos os dispositivos e servidores aqui apresentados são preparados para IPv6.

🔗 Blockchain

Estudamos e aplicamos blockchain para rastreabilidade de dados IoT e validação distribuída, usando tecnologias como Ethereum e contratos inteligentes.
